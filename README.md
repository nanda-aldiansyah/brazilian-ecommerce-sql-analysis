# Brazilian E-Commerce SQL Analysis
This repository contains the SQL scripts used for cleaning and analyzing the Brazilian E-Commerce Public Dataset from Olist (Kaggle). The queries include data preparation, revenue analysis, customer insights, and monthly performance tracking.
# Overview
The project includes:
- Data cleaning (handling missing values, fixing categories, formatting dates)
- Revenue analysis (total revenue, category performance, top products)
- Customer insights (top spenders, location-based patterns)
- Monthly trend analysis (seasonality and revenue fluctuations)
# Tools Used
- PostgreSQL
- DBeaver
- Power BI
